# C++與減千億
## 問題描述
減千億是個C\+\+初學者。第一次作業中，老師要他完成一道經典的$A+B$問題，題目敘述如下：

「輸入兩個$-10^{11}$以上、$10^{11}$以下的整數$A$、$B$，請輸出他們的和$A+B$。」

於是，自信滿滿的減千億著手嘗試解出這道題目，但他卻連範例測資都過不了，因而前來請求你的協助。

請撰寫一個程式，讀入$A$、$B$，輸出它們的和$A+B$。
## 輸入格式
首行有兩個正整數 $A, B\left(-10^{11}\le A, B\le 10^{11}\right)$。
## 輸出格式
輸出一個整數於一行，即它們的和$A+B$。
## 範例輸入1
```
1 1
```
## 範例輸出1
```
2
```
## 範例輸入2
```
1 -100000000000
```
## 範例輸出2
```
-99999999999
```
